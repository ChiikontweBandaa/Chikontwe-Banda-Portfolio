# Task Management Application

A full-stack PHP web application for managing tasks with user authentication.

## Features
- User registration and login
- Create, view, edit, delete tasks
- Task priorities and status tracking
- Secure user data isolation

## Setup
1. Place in XAMPP htdocs folder
2. Import database schema from phpMyAdmin
3. Access via http://localhost/task_manager
